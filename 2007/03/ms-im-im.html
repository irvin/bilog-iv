im 一下，真能做善事？

2007年3月10日 來源：blogger 原文連結

im 做善事，到底有甚麼意義？

第一：非使用 Windows Live Messenger 8.1 以上版本的人，他的聯絡人清單上會是一堆亂碼，半強迫他去升級8.1；而 Windows XP 以下的作業系統（兩千、NT…等）都沒有 WLM 8。im 一下，等於幫助微軟強迫你的朋友花錢升級作業系統。

第二：沒有使用 WLM 而是使用其餘可與 WLM 互通的軟體，包含蘋果上的 Adium、Linux 與 Windows 上的 Gaim 等相關軟體的人，聯絡人清單也會是一堆亂碼。im 一下，就成為懲罰你不是使用 XP 或 VISTA 的朋友。

第三：如果大家都因為考量到可以做公益，而更常使用 WLM 聊天，顯著提昇最多的，不會是微軟的捐款金額，而是 WLM 上的廣告顯示率。微軟在 WLM 上增加的收益，反而會比捐出來的多更多，而且還可抵稅。此外非美國使用者參與 im，也只是在幫微軟賺錢，沒有公益效果。

im 只是一種可怕的病毒式行銷，商業的宣傳手法，而且是很成功的一種。對我來說，這個活動只是讓我的 Adium 清單上一堆亂碼；當然有許多人會認為，沒有 WLM 8 可以用，或是反對這個活動，就不要用 WLM 就好。問題是，即時通訊市場被 WLM 寡佔的情形之下，說不用可能嗎？合理嗎？

im 行銷，就像是 Office 堅持使用封閉檔案格式一樣，當你傳遞 *.doc 或 *.ppt 檔給別人的同時，都是在幫微軟衝市佔率、都是在陷害買不起 Office 的人。微軟反對 ODF 格式的立場與原因大家都很明白，為的就是壟斷辦公室應用軟體所帶來的巨大利益。而從 im 這個活動中，也可看出其極大的商業考量。

總結一句：司馬昭之心。

（作者並不反對各位讀者 im 捐款做公益。作者所看不慣的是，微軟在這個行銷活動上，藉公益之名行賺錢之實，手法卻無比的拙劣）
