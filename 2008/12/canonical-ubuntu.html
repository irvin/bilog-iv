Canonical Store Ubuntu 官方商品

2008年12月17日 來源：blogger 原文連結

經歷訂單莫名其妙消失等阻礙，Ubuntu 紀念品終於越過半個地球到手了！內容物包含 Toomore 的毛帽 x2，我的預載 Ubuntu 8.10 4G 隨身碟、貼紙跟徽章各一，總價 35.74 英鎊。

等毛帽到 Toomore 手上，再請他應觀眾要求露面展示一下戴上的感覺……XD

本來要買的購物袋沒貨了，隨身碟沒標示出產地，外盒也什麼都沒有。原本還想說會不會是 MIT，XP 桌機插上有 Ubuntu 標記的隨身碟，今後會不會乖一點呢？

12/22補充：結果 VMware Fusion 偵測出來這顆是創見貨，不知道準不準就是了。

這隻隨身碟預載的內容其實是 Ubuntu 光碟，也就是使用 Ubuntu 8.10 中 系統 → 管理 → Creative a USB startup disk 功能所製作出來的。

此外這個功能有個缺點，製作時就要先劃分空間給 Ubuntu 隨身碟開機後使用。如果劃分 1GB，加上大約 1GB 的 Ubuntu，隨身碟本身就只剩 1GB 多一點可以裝東西了。我還以為 4G 很夠大了，看來是一點都不夠啊。

One more thing: 這顆隨身碟，沒有燈……
