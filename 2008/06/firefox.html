Firefox / IE7 宅配郵件追蹤搜索引擎

2008年6月9日 來源：blogger 原文連結

本文章已於 2009 / 5 更新嘍！請連到 Firefox 宅配郵件追蹤搜索引擎（更新）閱讀。

這是我自製的 Firefox 宅配追蹤搜索引擎（Search Plugin），提供給常常查詢宅配、掛號進度的您使用！
黑貓追蹤也可以在 IE7 中使用，另外兩個就沒有辦法了。安裝方式跟 Fx 相同。

開啟自動安裝頁面

本搜索引擎適用於 Firefox 2 與 Firefox 3，請點選上方「開啟自動安裝頁面」，開啟後搜索列圖示會變色，偵測出三個可自動安裝搜索引擎。如果沒有出現，請手動下載下列 XML 檔，放置於 Firefox 安裝目錄下的 searchplugin 目錄內即可使用。

台灣郵政郵件追蹤 (xml)
大嘴鳥宅配通貨物追蹤 (xml)
黑貓宅急便貨物追蹤 (xml)
