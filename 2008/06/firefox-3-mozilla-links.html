Firefox 3 下載日、Mozilla Links 、火狐鬥陣遊台灣徵人中

2008年6月19日 來源：blogger 原文連結

下載新火狐，手滑創紀錄的 下載日 結束了，目前 Firefox 3 全球下載次數已經超過 826 萬次，這個數字代表了，這一整天內，每秒鐘都有將近一百個人在下載安裝 Firefox 3，在推廣了這麼久之後，Firefox 能有這種成就，真是讓人非常興奮。

台灣最後正巧超過七萬的下載數，雖然比不過人多勢眾的中國，但是也超越了南韓，很爽。以台灣的人口數來說，每三百人僅有一個人在用 Firefox 3。這樣說起來雖然不多，但是根據我自己幾個網站 Google Analytic 得到的數據，其實 Firefox 在台灣大概有 10% 以上的市佔率了。經過今天，應該也再提昇了一些。

正體中文 Firefox 的基地是 Moztw，最近花了不少時間投入翻譯工作的 Mozilla Links 正體中文板 則是 Moztw 相關事業中最多即時報導、新消息的新聞 Blog，想要更加了解 Firefox，這是必看之站。下面推薦兩篇最近翻譯的文章：
Firefox 3 跟 2 有什麼差別？300 秒內完全搞懂 Firefox 3
想深入了解 Firefox 3？Firefox 3 釋出"前"預覽
（因為 Firefox 3 全解析 Mozilla Links 機動隊 還沒有翻譯完，這篇應該是最合適的文章了。

Firefox 2 上市時我們在台北信義誠品辦了一個人多到滿出來的歡樂 火狐大轟趴 2.0，3.0 有沒有呢？有、有、當然有，只是還在籌劃中……剛好卡到期末的時間，Moztw 人力實在很不足。有興趣幫任何忙的朋友歡迎到 Moztw-party group 來喊聲報名。

最後廣告一下：由我主辦的「火狐鬥陣遊台灣」，將會在七月出巡全台，同樣徵求台灣各地有志之士參加。本活動以歡樂為最高指導原則，我們要邊快樂的玩、邊向全世界狐迷們介紹台灣。這是 活動計畫、這是 徵人頁面，有興趣者請直接修改協調頁面內容即可。
