Post with Echofon bookmarklet / Echofon 推文工具

2010年3月6日 來源：blogger 原文連結

Recently I'm trying Echofon instead of Tweetie as the default Twitter client on my Mac, I'm satisfaction with its native support for list and official retweet, geolocation functions, but the UI and appearance of Tweetie was still much elegant than the Echofon does, although it was an year old software and havn't been update for several months. Another issue is that Echofon had no menu icon to inform the unread tweets.

One of the major functions for me of twitter is to bookmark and sharing the web page I'm reading, I'm get used with the "post to tweetie" bookmarklet to do so, I couldn't found one similar on Echofon, thus I'd made one out of Manu's for tweetie: [Post with Echofon]. You can select the text, or it will pick the page's title to post along with the URL.

這幾天我正在測試 Echofon，Mac OS X 上一個新的 Twitter Client（用戶端程式）。過去受許多 Mac 用戶喜愛的 Tweetie 雖然有非常優雅的 UI，但是對於 Twitter 的打星星、List、官方 RT、地理資訊（Geolocation）等新功能的支援卻非常不足，在此方面 Echofon 的表現則極為優異。

在使用 Tweetie 時，我很習慣使用 Bookmarklet 來推文，但是遍尋不著 Echofon 專用的，因此參考了 Manu 製作的 Tweetie Bookmarklet 修改了一個，在此分享給各位推友：[Post with Echofon]（直接拖拉連結至書籤列即可）。
