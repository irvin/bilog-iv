name="fc4d" class="section section--body"><div class="section-divider"><hr class="section-divider"></div><div class="section-content"><div class="section-inner sectionLayout--insetColumn"><h3 name="6c6f" id="6c6f" class="graf graf--h3 graf--leading">發言內容</h3><blockquote name="e5bf" id="e5bf" class="graf graf--blockquote graf-after--h3">（經潤飾編整）</blockquote><p name="a4a6" id="a4a6" class="graf graf--p graf-after--blockquote">副局長、議員、老師及各位朋友好，我是成大交管所，鄭永祥老師研究室碩士畢，台灣維基媒體協會專員及開放文化基金會技術顧問。</p><p name="fe04" id="fe04" class="graf graf--p graf-after--p">今天我的意見主要在城區段的部分。</p><p name="f2c5" id="f2c5" class="graf graf--p graf-after--p">回顧十年前，台南市選擇高架跨座式單軌系統，且市民支持的理由是：城區路幅狹小，因而有轉彎半徑的限制，且避免高架軌道的量體，對古蹟與市民生活造成壓迫。</p><p name="f5a9" id="f5a9" class="graf graf--p graf-after--p">然而，現在的規劃決定在舊城區採取地下化方式，這是否意味著當初選擇造價較高的單軌系統，理由已不存在？</p><p name="da95" id="da95" class="graf graf--p graf-after--p">跨座式單軌地下化，由於隧道半徑的問題，施工造價甚至可能比傳統鋼軌或膠輪中運量系統地下化來的更高。</p><p name="c84b" id="c84b" class="graf graf--p graf-after--p">今天的簡報中，未看到針對台南地質條件的潛盾施工評估，以及<strong class="markup--strong markup--p-strong">地下化與非地下化的成本效益的差異評估</strong>，也可能在送院審查的過程中產生不確定性與風險。</p><p name="3582" id="3582" class="graf graf--p graf-after--p">雖綠線會建設自己的機場，也不會跟藍線連通運轉，未必需要堅持成本更高的跨座式單軌系統。但由於藍線已接近動工，且單軌較符合市民對於捷運系統兼有觀光效用的想像，我仍然支持維持單軌系統。</p><p name="9686" id="9686" class="graf graf--p graf-after--p">然而，如果今天是考量<strong class="markup--strong markup--p-strong">城區路幅不足</strong>與<strong class="markup--strong markup--p-strong">轉彎半徑</strong>的問題，因而決定採取造價高昂，且有諸多疑慮的地下化，故選線又未進入城區的核心，我認為其實仍有更好的方案。</p><h3 name="ffbe" id="ffbe" class="graf graf--h3 graf-after--p"><strong class="markup--strong markup--h3-strong">高架上下行路線</strong>分流</h3><p name="2990" id="2990" class="graf graf--p graf-after--h3">我建議台南捷運在老城區的選線，應考慮東西向、上下行分流的路線規劃。</p><p name="bfd2" id="bfd2" class="graf graf--p graf-after--p">例如西行路線可行經中山路、民生路、民族路，東行則經成功路；又或者，上行經成功路或民生路接海安路，下行維持今天的規劃，即中山路、中正路、南門或忠義路接府前路。</p><p name="ac1b" id="ac1b" class="graf graf--p graf-after--p">維持高架設計並分流路線的優勢包括：</p><ol class="postList"><li name="02d8" id="02d8" class="graf graf--li graf-after--p"><strong class="markup--strong markup--li-strong">縮短地下化施工的工期</strong>：目前地下車站單邊施工開挖至少需三年，整體地下段工期七至八年，若維持高架設計，可大幅縮短工期。</li><li name="666c" id="666c" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">降低整體建設成本，有效運用資金</strong>：省下地下工程所需的高昂經費，可將節省的資源用於路線延伸，如更加深入核心的城區，或像今天多位議員提的直接延伸到歸仁等方向。</li><li name="d342" id="d342" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">減少地下施工對居民與古蹟的風險</strong>：避免因地下化挖掘到古蹟或文化層，造成長期延宕。</li><li name="cb95" id="cb95" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">深入城區擴大服務範圍</strong>：分兩線深入城區，可涵蓋更廣的區域、更多店家與主要觀光景點，紓解現行民族、民權、成功、海安、西門等路段例假日的交通壅塞，改善公共運輸在城區對觀光的服務水準。</li><li name="2143" id="2143" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">避免地下段出土的成本與問題</strong>：因為要下到地面下，反倒增加了東西兩個出土路段造成的交通阻礙與選址等問題。</li><li name="2b9c" id="2b9c" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">配合上下行軌道，一併改善橫向巷道環境</strong>：如上下行軌道選在相鄰的兩路，如民權民生、府前中正，建設中可一併將橫向的巷道進行整體人行環境的改善。</li></ol><p name="78b3" id="78b3" class="graf graf--p graf-after--li graf--trailing">希望交通局及顧問公司，除地下化以外，也能將高架上下行分流深入城區的替代方案納入考慮，比較兩者的成本效益。不但能讓城內的公共運輸服務範圍更加擴大，提升市民與觀光客的交通體驗，也才符合市民當時對單軌系統的想像與期待。</p></div></div></section><section name="de0e" class="section section--body
