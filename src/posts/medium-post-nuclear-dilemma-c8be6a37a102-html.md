<section name="663a" class="section section--body section--first"><div class="section-divider"><hr class="section-divider"></div><div class="section-content"><div class="section-inner sectionLayout--insetColumn"><h3 name="d709" id="d709" class="graf graf--h3 graf--leading graf--title">核電公投到底是在同意什麼？</h3><p name="93dc" id="93dc" class="graf graf--p graf-after--h3">週末要去投同意核電的人，真的要好好想一下自己到底在投什麼勒：</p><h3 name="9053" id="9053" class="graf graf--h3 graf-after--p">a. 重啟核三</h3><p name="fdff" id="fdff" class="graf graf--p graf-after--h3">核電廠不是電腦，不是按一個開關就重開機了欸。</p><p name="1a24" id="1a24" class="graf graf--p graf-after--p">得先花幾百億檢查跟更新設備，再買新的燃料棒進來，可能五年後可以順利啟動開始發出第一度電，結果剩餘的燃料池空間也只能再撐不到五年。</p><p name="86c9" id="86c9" class="graf graf--p graf-after--p">想要再繼續用下去，得要有乾儲可以退出池中的燃料棒。用了四十年的機組總共需要一百座以上的乾儲，先去問哪個縣市長會核下去？核一乾儲蓋到現在，新北也才核准六座，後面還欠兩百五十座欸。</p><p name="b31d" id="b31d" class="graf graf--p graf-after--p">核三幾百億花下去，只能用不到五年，要怎麼折舊？到時候一度要多少錢？</p><p name="9ed7" id="9ed7" class="graf graf--p graf-after--p">還是我們未來的新北市長決定全部承擔下來，南部民進黨縣市不蓋的乾儲全部蓋到新北來，新北市民補償金年年領到飽！反正也就三四百座兩個人高的水泥核廢料槽而已，只要能打房，作為新北市民我舉雙手贊成啦！</p><h3 name="8b1e" id="8b1e" class="graf graf--h3 graf-after--p">b. SMR</h3><p name="8a0c" id="8a0c" class="graf graf--p graf-after--h3">SMR 不是只是小顆的反應爐，SMR 是小型「模組化」反應爐，是像潛艦、航母上的反應爐一樣，一大台包起來整個自洽，拖車拖過來插上去就能發電，用幾年燃料燒光了，就整個換掉拖回去埋。</p><p name="c51a" id="c51a" class="graf graf--p graf-after--p">現在世界上只有中國跟它俄爹有已經號稱開始商轉的模組，實際上等到美國公司弄出來，台灣下單到可以到貨，少說也是十年起跳的期程。</p><p name="d9bc" id="d9bc" class="graf graf--p graf-after--p">然後你拿到手的一顆的成本跟現在一座機組差不多，發電量只有幾分之一。傳統核電要 20 座才夠台灣用電，SMR 可能要超過兩百座。</p><p name="a985" id="a985" class="graf graf--p graf-after--p">假設一座可以燒五年，每年要換掉 40 座，我就請問每年增加 40 台比乾儲還大顆的 SMR 屍體要堆到哪去？每個縣市都要裝 20 台 SMR 地點想好了嗎？然後台電每年要換 40 個機組的費用，到底攤提下來一度要多少錢？</p><p name="ca13" id="ca13" class="graf graf--p graf-after--p">鄉親真的喜歡自家旁邊的那顆反應爐，還每五年要大施工？我是覺得乾脆以後每一隻中華電信的基地台都附帶一顆 SMR 啦，地震不怕沒電發抖音！</p><h3 name="7039" id="7039" class="graf graf--h3 graf-after--p">c. 第四代核電</h3><p name="0639" id="0639" class="graf graf--p graf-after--h3">第四代核電號稱可以「燒掉」現有的高階核廢料，問題就不是燃料池打開核廢料拿起來，丟進去就可以燒了勒。</p><p name="b3c5" id="b3c5" class="graf graf--p graf-after--p">就算 2030 真的有機組可以商轉了，到時也是要 1) 跟美國爸爸買反應爐並且蓋好 2) 把核廢花錢出口給美國爸爸處理 3) 再花錢跟美國爸爸買他們處理好的燃料回來用⋯⋯</p><p name="1011" id="1011" class="graf graf--p graf-after--p graf--trailing">先不管 20 年內會不會開始發電啦，有想好整個再處理的燃料成本是多少錢了嗎？到時就不要有人又出來咆哮電費太貴台電虧損 blablabla</p></div></div></section><section name="d8ab" class="section section--body section--last"><div class="section-divider"><hr class="section-divider"></div><div class="section-content"><div class="section-inner sectionLayout--insetColumn"><p name="f25e" id="f25e" class="graf graf--p graf--leading">不論是哪一種方案，就都不是三五年能發得出來的事。真要拿這個來確保能源安全，還沒等到發電，中共就已經統一台灣了。</p><p name="a7b7" id="a7b7" class="graf graf--p graf-after--p">所以發起公投的黃國昌，有告訴大家這個同意投下去是五年起跳，政府錢就要開始燒了，然後到下次大選你還用不到半度核電嗎？</p><p name="8bf8" id="8bf8" class="graf graf--p graf-after--p">所以你投的同意票，到底是要同意哪一種核電？</p><p name="69df" id="69df" class="graf graf--p graf-after--p graf--trailing">真的考慮好不論未來成本變多高，都要硬著頭皮用下去了嗎？</p></div></div></section>
