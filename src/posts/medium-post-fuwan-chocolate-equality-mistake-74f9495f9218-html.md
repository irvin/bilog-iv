name="a3fc" id="a3fc" class="graf graf--h4 graf--leading">「公關危機處理」無法洗清品牌負評</h4><p name="a1f6" id="a1f6" class="graf graf--p graf-after--h4">福灣巧克力的性騷擾事件持續延燒。傳統的公關危機處理 DISCO，原則上會在第一時間，界定出事件關係人，以行動與溝通雙軌方式，公開承諾與作為，試圖控制輿論發展，進行快速止血。</p><p name="5177" id="5177" class="graf graf--p graf-after--p">但網路時代，當你的品牌已被大眾蓋上負面印記，以一兩次口頭或書面的道歉、物質補償，希冀能在短期內止息風波，讓大眾淡忘過錯，是完全不切實際的想法。</p><p name="b021" id="b021" class="graf graf--p graf-after--p">當道歉再多次，也無法讓大眾接受到誠意，任何行銷、回饋，都不足以消止抵制時，錯誤一次次的被提起，企業如何表示真誠的反省，來求取大眾的原諒？</p><h3 name="c275" id="c275" class="graf graf--h3 graf-after--p">超越預期的持續直接行動</h3><h4 name="bfda" id="bfda" class="graf graf--h4 graf-after--h3">直接之行動</h4><p name="5460" id="5460" class="graf graf--p graf-after--h4">事件當事人需誠實直接的面對錯誤核心，針對所犯的過錯，以實務行動去進行彌補，對職場平權領域做出直接的貢獻。</p><h4 name="99ca" id="99ca" class="graf graf--h4 graf-after--p">超越預期之水準</h4><p name="ef96" id="ef96" class="graf graf--p graf-after--h4">不但需以實際作為表示歉意，且必須要超乎大眾的預期，以國際一流平權職場之水準為目標。</p><h4 name="e540" id="e540" class="graf graf--h4 graf-after--p">長久且持續</h4><p name="d188" id="d188" class="graf graf--p graf-after--h4">以超乎大眾預期，且長期持續的行動，來扭轉大眾對品牌的觀感</p><h3 name="58de" id="58de" class="graf graf--h3 graf-after--p">福灣彌補性別平權之行動方案</h3><p name="9710" id="9710" class="graf graf--p graf-after--h3">在此針對福灣作為國際品牌、產業領導廠商之角色，擬定職場性別平權之彌補建議行動項目：</p><h4 name="d31b" id="d31b" class="graf graf--h4 graf-after--p">1. 定期、持續捐助職場性騷擾防治</h4><p name="affe" id="affe" class="graf graf--p graf-after--h4">以每年至少一百萬（受害者的原始求償金額），持續捐助職場平權組織與相關計畫。以實際金錢支付公益，對性騷擾與毀謗兩案，表示「雖當時法院判決撫慰金僅15萬，但時至今日，願意回過頭全力彌補」一意。</p><p name="2b56" id="2b56" class="graf graf--p graf-after--p">捐助對象應針對職場平權倡議計畫，如同志諮詢熱線及婚姻平權大平台之「<a href="https://hotline.org.tw/news/2869" data-href="https://hotline.org.tw/news/2869" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">友善職場指南</a>」（ <a href="https://hotline.org.tw/news/2869" data-href="https://hotline.org.tw/news/2869" class="markup--anchor markup--p-anchor" rel="nofollow noopener noopener" target="_blank">https://hotline.org.tw/news/2869</a> ）或女人迷之「<a href="https://womany.net/gia/2018/diversity-and-inclusion" data-href="https://womany.net/gia/2018/diversity-and-inclusion" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">多元共融與實踐</a>」（<a href="https://womany.net/gia/2018/diversity-and-inclusion" data-href="https://womany.net/gia/2018/diversity-and-inclusion" class="markup--anchor markup--p-anchor" rel="nofollow noopener noopener" target="_blank">https://womany.net/gia/2018/diversity-and-inclusion</a>）倡議。</p><h4 name="b3e6" id="b3e6" class="graf graf--h4 graf-after--p">2. 設置企業平權長，建立多元平等包容制度</h4><p name="2b65" id="2b65" class="graf graf--p graf-after--h4">在企業內部設置平權長（Chief Diversity Officer, CDO），聘請專業人員，建立保障職場多元、平等、包容（Diversity Equality &amp; Inclusion, DEI）三方面之內控措施及持續精進制度。</p><h4 name="a974" id="a974" class="graf graf--h4 graf-after--p">3. 提出完整事件調查與檢討報告</h4><p name="5370" id="5370" class="graf graf--p graf-after--h4">以 <a href="https://en.wikipedia.org/wiki/Postmortem_documentation" data-href="https://en.wikipedia.org/wiki/Postmortem_documentation" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Postmortem 格式</a>提出檢討報告，由平權長協助事件關係人與企業內各級成員，以時間為軸、細節為體，自我揭露兩起事件之發生過程與原因、影響，闡明如何避免未來再次出錯。</p><h4 name="8112" id="8112" class="graf graf--h4 graf-after--p">4. 每年釋出 DEI 多元報告，行動擴及產線上下游廠商</h4><p name="d541" id="d541" class="graf graf--p graf-after--h4">承諾每年釋出 DEI 報告，內容涵括職場多元平等包容之現況數據、目標與實際行動計畫。並以產業領導品牌之角色，將報告範圍擴及產線上下游，含契作農園與下游通路。一併設定目標，提出行動方案，並協助協力廠商改善相關指標。</p><h3 name="6cf2" id="6cf2" class="graf graf--h3 graf-after--p">成為平權議題之業界標竿</h3><p name="eaef" id="eaef" class="graf graf--p graf-after--h3">如果福灣巧克力能以實際行動持續付出，努力成為業界之平權標竿，甚至能發揮影響力，讓屏東巧克力產業成為最為多元包容的職場，從業人員與巧克力的口味一樣多元豐富，相信消費者都能看到付出與誠意。</p><p name="97d5" id="97d5" class="graf graf--p graf-after--p">讓品牌從錯誤中成長茁壯。如果未來，所犯的過錯每次被回憶起時，相對的彌補作為都能一併在旁，福灣巧克力才能重拾消費者的信心與支持。</p><figure name="32d5" id="32d5" class="graf graf--figure graf-after--p
