今天在蘋果日報上面看到這則消息，剛剛逛Blog時也看到udn報出來。<br />
<blockquote>中正大學發表Web3.0 最快明年3月上市‧聯合新聞網 2006/12/28 (<a href="https://flic.kr/p/vHMi5">報導抓圖</a>)<br />
<a href="http://mag.udn.com/mag/it/storypage.jsp?f_ART_ID=55294">http://mag.udn.com/mag/it/storypage.jsp?f_ART_ID=55294</a><br />
最近一兩年網路颳起Web 2.0的旋風，但在台灣媒體與網路界還正在熱切的討論與「擁抱」Web 2.0的時候，國立中正大學今天發表網路資訊系統Web 3.0計劃，強調將提供一個網路民主、均富、平權的時代，預定最快明年三月上市。</blockquote><br />
所以，這個web 3.0到底是什麼東西？因為 <a href="http://www.oreillynet.com/pub/a/oreilly/tim/news/2005/09/30/what-is-web-20.html">O'Relly</a> 也還沒辦法告訴我什麼是Web 3.0，我只好去問<a href="http://www.hemidemi.com/search/bookmark?keyword=web+3.0">黑米大神</a>...<br />
<br />
逛了一堆黑米上的Web 3.0相關網站，我還是沒有看到什麼特別的東西，只好回過頭來看這篇新聞。想想，Web3.0耶！2.0的風潮都還方興未艾，台灣居然搶先全世界提出Web3.0的概念。還是認真讀報導的內容吧。<br />
<div id="fullpost"><br />
<blockquote>計劃主持人國立中正大學網際網路研究中心主任吳昇副教授說，以網路文明歷史的宏觀來看，目前網路處於商業力量高度主宰的資訊帝國時代，網路文明集中在一些超級大站如Google、Yahoo!、MSN等超強業者，頂尖大學的網路文明程度仍遠不如那些大站，更遑論一般團體或個人，網路活動有被超級大站「壟斷」的隱憂。<br />
<br />
吳昇說，Web 3.0時代網路公民將被賦予基本的資訊權，一個在網路世界耕者有其田，住者有其屋的基本資訊人權，使用者可當家做主、輕鬆在自己的電腦架設個人或家庭入口網站與部落格，進行更方便、更有效率的社群活動、資訊服務與分享；學校等社群團體將可免費擁有與頂尖的入口網站同等級的軟體系統，建構具高度吸引力的社群入口網站服務網民。</blockquote>
<br />
所以，Web 3.0，也就是他們的「NUWeb」，是要提供<span>「一套簡單好用的社群、入口網站系統」</span>，以讓使用者用來挑戰（至少是擺脫）Google（Blog）或Youtube（Video）這些Service Provider。<br />
<br />
<blockquote>他指出，在Web 3.0的超大型入口網站與社群網站，是一個互助合作的分散式入口網站，使用者擁有幾乎無限制的使用空間、更先進的搜尋、分享與知識管理功能，整個社群將由所有使用者共同參與、共同分享、共同擁有、共同治理，就是最理想的網路民主新時代。
</blockquote>
<br />
但是下一段，他又提到要建立一個Web 3.0的超大型入口網站。這不是上一段指出要挑戰的東西？「分散式」似乎是期中的關鍵字。所以這是個「P2P式的社群、入口網站系統」，這樣看就有點新意了。<br />
<br />
<blockquote>為追求Web 3.0的理想境界，吳昇兩年前帶領新典資訊的研發團隊與國立中正大學GAIS實驗室開始研發一個名為NUWeb的網路資訊系統。它是以使用者為中心的分散式入口網站與資訊分享平台系統，包含NUWeb.cc網路中心、社區入口網站、與個人入口網站等三部分。<br />
<br />
吳昇指出，NUWeb.cc就像YouTube或Blog資訊分享網站，讓大家分享交流，不同的是空間幾乎沒有限制、使用者有充分自主權，整個社群由大家共同擁有、共同治理。NUWeb PP則讓使用者輕鬆的在自己的電腦架設個人或家庭入口網站與部落格，利用自己的分享平台來組織管理分享的目錄，且可透過PC對PC直接連結和親朋好友不受限的分享相片影片等資訊。</blockquote>
<br />
這裡又出現了三項東西：第一個是NUWeb的入口網站、第二個是社區入口網站、第三個是個人入口網站，所以整個計畫又變成了一個很大的入口網站。再加上一個讓別人從自己電腦上抓東西的軟體……<br />
<br />
<blockquote>NUWeb是一個群策群力的公益計畫，正式發行時將開放原始碼，目前正在積極尋求更多合作團隊。</blockquote>
<br />
好吧，就算整個報導再讓人看不懂一點，就算NUWeb（個人簡單架站分享檔案）有沒有什麼超越web 2.0概念上（使用者創造內容）的突破，就算這東西再<a href="http://2tigers.net/blog/2006/06/02/web-30-probably-more-creative/">消費Web 2.0</a>一點，至少它會是Open Source的。</div>
