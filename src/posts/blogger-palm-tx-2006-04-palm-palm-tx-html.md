<h2>Browser 瀏覽器</h2>

<h3>TX 上的瀏覽器 Blazer</h3>
<p>我最常使用 TX 來上的兩個網站之一，PALMisLIFE 可攜版。可攜版的 PIL 只有文字沒有圖，也把表格都給拿掉了，很適合用 TX 來爬文。只是希望有個切換圖片顯示的功能，否則看圖還要切到完整版去，又會被表格干擾。</p>

<a href="http://www.flickr.com/photos/irvin/124213503/" title="Flickr 上 Irvin Chen 的 Blazer看PALMisLIFE可攜版"><img src="http://farm1.static.flickr.com/45/124213503_14f67a463f_m.jpg" width="163" height="240" alt="Blazer看PALMisLIFE可攜版" /></a>

<h2>BBS</h2>

<p>相信有很多朋友跟我一樣還是學生，這樣的話應該也對 BBS 不陌生。BBS 是華人特有的一種網路文化，結合了資訊交流、即時通訊、交友等各種功能於一身，純文字與零滑鼠的介面也是一種特色。</p>

<p>在 Windows 下，有 <a href="http://pcman.ptt.cc/">PCMan</a> 這套方便的程式可以上 BBS；在 Mac OS X 及 X Window 也有跨平台的 <a href="http://pcmanx.csie.net/">PCMan X</a> 可以使用，可惜因為中文編碼的關係，目前在Palm上面還沒有可以正常使用來上 BBS 的 telnet 軟體。</p>

<h3>pssh</h3>

<p>不過總是有需求就會有方法可以滿足。台灣最大的 BBS <a href="http://www.ptt.cc/">Ptt</a> 特別提供的 SSH 連線，本是提供網友加密上線用的，反倒可以讓我們在 TX 上利用 pssh 登入。 </p>

<a href="http://www.flickr.com/photos/irvin/124213149/" title="Flickr 上 Irvin Chen 的 pssh上ptt 讀文"><img src="http://farm1.static.flickr.com/35/124213149_776ea149ec_m.jpg" width="240" height="162" alt="pssh上ptt 讀文" /></a>

<p>不但中文正常，連顏色都正常，還有銀幕的方向鍵可以直接點，TX 的 wi-fi，是不是就該這樣用呢？</p>

<p>Ptt 使用 SSH 連線的設定方式，另外 Ptt2 也可以用同樣的方法連線。</p>

<a href="http://www.flickr.com/photos/irvin/124212956/" title="Flickr 上 Irvin Chen 的 PSSH上PTT的設定"><img src="http://farm1.static.flickr.com/53/124212956_48e0a213fc_m.jpg" width="240" height="172" alt="PSSH上PTT的設定" /></a>
   
<p>pssh 連上 Ptt 的進站畫面，圖雖然有點破碎，不過還是可以正常登入的。</p>

<a href="http://www.flickr.com/photos/irvin/124213351/" title="Flickr 上 Irvin Chen 的 pssh上ptt 進站畫面"><img src="http://farm1.static.flickr.com/50/124213351_99d1dd5c93_m.jpg" width="240" height="164" alt="pssh上ptt 進站畫面" /></a>

<p>pssh 的 Options 選單，可以切換 title bar 及 control bar（TX 的灰色工作列）顯示與否及小鍵盤的型式。</p>

<a href="http://www.flickr.com/photos/irvin/124213253/" title="Flickr 上 Irvin Chen 的 pssh Option選單"><img src="http://farm1.static.flickr.com/46/124213253_6bb0a55003_m.jpg" width="240" height="161" alt="pssh Option選單" /></a>
