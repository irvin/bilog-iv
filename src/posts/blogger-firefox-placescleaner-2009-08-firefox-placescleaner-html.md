<p>經由 littlebtc 的協助，昨晚我們將 <a href="/2009/08/firefox-3.html">Firefox 減肥程序</a> 寫成了「<a href="https://addons.mozilla.org/zh-TW/firefox/addon/13860/">PlacesCleaner 收藏庫清潔工</a>」套件，歡迎大家幫忙測試。</p>

<p>以下是簡單教學，如果使用上遇到任何狀況、意見或建議，歡迎 <a href="http://forum.moztw.org/viewtopic.php?f=11&t=27767">到 MozTW 論壇回文</a>。</p>

<p>安裝由此去 → <a href="https://addons.mozilla.org/zh-TW/firefox/addon/13860/]PlacesCleaner">收藏庫清潔工 :: Firefox 附加元件</a></p>

<p>功能：<br />
1. 清除瀏覽歷史中瀏覽次數不超過2次的紀錄（可自行設定瀏覽次數限制）<br />
2. 清除部份問題版本 Google Toolbar 紀錄的網頁縮圖，處理引起的收藏庫肥大<br />
3. 壓縮重整收藏庫</p>

<p>使用方法：</p>

<p>裝好後，右下角的掃把按下去就完成了。<br />
<a href="http://www.flickr.com/photos/irvin/3848073701/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.1 - 1"><img src="/assets/images/farm4.static.flickr.com/8a04306b65399b42.jpg" width="500" height="427" alt="PlacesCleaner 0.1 - 1" /></a></p>

<p>透過掃把圖示的右鍵選單可進入偏好設定<br />
<a href="http://www.flickr.com/photos/irvin/3848073995/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.1 - 2"><img src="/assets/images/farm3.static.flickr.com/87bb8b420d2c607a.jpg" width="500" height="410" alt="PlacesCleaner 0.1 - 2" /></a></p>

<p>偏好設定唯一的選項，用來設定清除瀏覽次數少於幾次的資料（預設值是2次）<br />
<a href="http://www.flickr.com/photos/irvin/3848866232/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.1 - 3"><img src="/assets/images/farm3.static.flickr.com/55144d9f68a79fa4.jpg" width="500" height="411" alt="PlacesCleaner 0.1 - 3" /></a></p>
