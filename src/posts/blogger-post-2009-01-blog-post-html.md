<p>過了一個多月，現在來回顧一下去年 (2008) 年底忙了兩個月事：高雄市職業總工會網路教學課程。</p>

<p>這個課程源自於高雄市職業總工會的郭先生，郭先生先是聯絡 Mozilla Taiwan 的柏強提出構想，想開一堂 Firefox 辦公室應用的課程，後來由身在南部的我跟 Toomore 承接下來。</p>

<p>原始構想雖然是 Firefox 辦公室應用課程，但是一堂課 3 小時，整個課程總共 16 堂課，我們想了過來又想了過去，不知道對職業公會的學員們怎麼講 Firefox 講到 48 小時……因此就變成這樣的課程了：從最簡單的上網申請帳號開始，以各式各樣的網路應用為主，自由軟體為輔的一堂課。</p>

<p>我們對課程的想法是，要讓原先已經會打開電腦、打字、使用 IE 的學員，能了解像我們這樣（7 年級，大概是學員小孩年紀）的年輕人，在網路上都在玩些什麼，至少讓學員知道網路能做的事，並且在生活上多少用得上。</p>

<div id="fullpost"><p>值得一提的是，因為上課時大概都能有兩到三位朋友到場（曾經參加的有 wini、凍人翔、Toomore 女朋友）， 因此除主講者外，學員在操作時，都能有人從旁協助，讓課程順利不少。</p>

<p>下面就是 9/15 到 11/10 實際講授的課程，包含課程講義跟投影片，都放在 Google Docs 跟 Slideshare 與大家分享。</p>

<p>第 3 堂課 9/22</p>

<u>電子郵件</u><br />
Gmail、Twitter、新酷音輸入法<br />
講義：<a href="http://docs.google.com/Doc?id=dhm4tqmm_118dv4vv7g6">http://go2.tw/class03</a><br />
<p>系列課程中有幾堂課，因為我的時間關係來不及作投影片，這堂就是其中之一。第三堂課講的是網路最基本的應用 email。學員跟我們想像的一樣，都只知道 yahoo 的 email，因此課程著重在 gmail 的優點跟 gmail 內建的 gtalk 上。本堂課讓學員互相寄信、使用 gtalk 傳訊，是很有效果的實作之一。此外課程中展示了一下 twitter，雖然我認為 twitter 很適合學員與講師間的意見交流，想讓大家實際使用，但是有鑑於申請 Google 帳號時的困難、跟時間不夠，因此只作展示。新酷音輸入法也是只展示沒有讓大家實際安裝。</p>

<p>第 4 堂課 9/24 &amp; 第 5 堂課 10/1</p>
<u>Picasa 網路相簿</u><br />
講義：<a href="http://sites.google.com/a/ice9s.org/publicfiles/files/%E5%8D%97%E5%8D%80%E5%8B%9E%E5%B7%A5%E8%AA%B2%E7%A8%8B-%E7%B6%B2%E8%B7%AF%E7%9B%B8%E7%B0%BF.pdf?attredirects=0">http://go2.tw/class04</a><br />

<u>Picasa 相簿軟體</u><br />
講義：<a href="http://sites.google.com/a/ice9s.org/publicfiles/files/%E5%8D%97%E5%8D%80%E5%8B%9E%E5%B7%A5%E8%AA%B2%E7%A8%8B-%E7%B6%B2%E8%B7%AF%E7%9B%B8%E7%B0%BFPicasa%E7%9B%B8%E7%89%87%E7%AE%A1%E7%90%86%E8%BB%9F%E9%AB%94.pdf?attredirects=0">http://go2.tw/class05</a><br />

<p>這兩堂課由 Toomore 講授。大家家中都有數位相機，也會按快門拍照，但是會把相片存入電腦、管理、然後上傳到網路相簿的人就很少了。因此這兩堂課要讓學員透過操作簡單的 Picasa 跟 Picasa Web Album 進行相片管理跟使用網路相簿。我對課程的原始構想是，要求大家攜帶相機、連接線或記憶卡來，第一堂課讓學員使用 Picasa 把相機中的相片讀入電腦，並學習整理與後置相片，第二堂課讓學員透過 Picasa 的同步功能直接把相片放上 Picasa Web Album。</p>

<p>實際進行的狀況是，Toomore 把兩堂課的順序倒過來教，先講網路相簿、再講 Picasa，結果在第一堂課就遇到許多問題。雖然 Toomore 做了兩份很詳細的講義，但是對學員來說，光是把相機接上電腦，將相機或讀卡機（我們準備了三台還是不夠）中的相片抓到桌面的資料夾，就觸礁了。因此每一步都得 Step-by-step 來進行。對於我們來說很簡單基本的檔案複製貼上、透過網頁介面上傳等動作，對於學員都極為困難。</p>

<p>如同我原始的想法，第二堂課時，我們教導學員使用 Picasa 匯入相機中的相片，進行後置、上傳跟 email 寄出（這個功能對學員來說很實用，雖然我們很少使用），整個流程跟學習效果，相較於第一堂課都好多了。驗證了我的看法：不必讓學員知道像 Picasa 這樣 UI 包裝很完美的軟體背後的流程與原理，只要知道怎麼使用能達到目的就好（同樣的概念也可以放在 iPhoto 上）。</p>

<p>其實相簿跟 Blog 這兩堂課，我很想教 Pixnet，但是有鑑於使用 Google 帳號跟使用 Picasa，因此只好忍痛放棄了</p>
