### Mozilla 透過開源碼支援專案投資支持 Python 生態圈

**MOSS Q4: Supporting the Python Ecosystem - The Mozilla Blog**
*Mozilla was born out of, and remains a part of, the open source and free software movement. Through the Mozilla Open…*blog.mozilla.org

2017 年 Q4 的 Mozilla 開放原始碼支援專案（Mozilla Open Source Support program, MOSS）獲選者包含 Python 生態圈中重要的 [Buildbot](https://buildbot.net/) 與 PyPI。其他獲選則包含 [Harfbuzz](https://freedesktop.org/wiki/Software/HarfBuzz/)、[Zappa](https://blog.zappa.io/)、[Tatoeba](https://tatoeba.org/)、Tor 專案的 [Open Observatory of Network Interference](https://ooni.torproject.org/) 與 [Commento](https://github.com/adtac/commento) 等專案獲得資助。

MOSS 專案仍持續接受提名中，另外也正接受開源碼專案申請安全檢核服務，詳情請查閱： [https://www.mozilla.org/en-US/moss/](https://www.mozilla.org/en-US/moss/)
