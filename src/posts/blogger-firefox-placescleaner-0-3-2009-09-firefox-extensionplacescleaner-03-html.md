<p>Firefox 套件 PlacesCleaner 收藏庫清潔工 0.3 版已經釋出，以下是簡單的使用教學。如果使用上遇到任何狀況、意見或建議，歡迎 <a href="http://forum.moztw.org/viewtopic.php?f=11&t=27767">到 MozTW 論壇回文</a>。</p>

<p>下載：<a href="https://addons.mozilla.org/zh-TW/firefox/addon/13860">PlacesCleaner 收藏庫清潔工 :: Firefox 附加元件</a></p>

<p>功能簡介：</p>

<p>1. 清除瀏覽歷史中瀏覽次數不超過2次的紀錄（可自行設定瀏覽次數限制）<br />
2. 清除部份問題版本 Google Toolbar 紀錄的網頁縮圖，處理引起的收藏庫肥大<br />
3. 壓縮重整（Vacuum）收藏庫</p>

<p>更新項目：</p>

<p>1. 不清理、只Vacuum的選項<br />
2. 手動模式下，清除後隱藏狀況列圖示數日<br />
3. 重新設計設定視窗<br />
4. 新增語言：fr, pl-PL, pt-BR, pt-PT, ru-RU</p>

<div id="fullpost"><p>安裝步驟：</p>

<p>1. 請由此去 <a href="https://addons.mozilla.org/zh-TW/firefox/addon/13860/">→ 收藏庫清潔工 :: Firefox 附加元件</a><br />
2. 選取中央「讓我安裝此實驗中附加元件」，再點選「安裝到 Firefox」</p>

<a href="http://www.flickr.com/photos/irvin/3884119434/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.3"><img src="http://farm3.static.flickr.com/2508/3884119434_0ba1ee6f32.jpg" width="500" height="362" alt="PlacesCleaner 0.3" /></a>

<p>3. 在套件安裝畫面中，等待倒數五秒後點選安裝按鈕，重開 Firefox 後就安裝完成了！</p>

<a href="http://www.flickr.com/photos/irvin/3884119610/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.3 - 2"><img src="http://farm3.static.flickr.com/2428/3884119610_fb09e01dc7_o.png" width="511" height="347" alt="PlacesCleaner 0.3 - 2" /></a>


<p>使用說明：</p>

<p>1. 直接點選右下角的掃把，清掃完成！</p>

<a href="http://www.flickr.com/photos/irvin/3884136790/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.3 - 3"><img src="http://farm4.static.flickr.com/3504/3884136790_7f1ccf2684.jpg" width="500" height="374" alt="PlacesCleaner 0.3 - 3" /></a>

<p>2. 掃把圖示上按右鍵→偏好設定，可以進入選項視窗，可選擇手動清理或自動清理。</p>

<a href="http://www.flickr.com/photos/irvin/3884119708/" title="Flickr 上 Irvin Chen 的 PlacesCleaner 0.3"><img src="http://farm4.static.flickr.com/3442/3884119708_e73532c759_o.png" width="450" height="268" alt="PlacesCleaner 0.3" /></a>

<p>自動清理的情況下，預設值是每 30 天在開啟 Firefox 時會清理收藏庫（作太多無益、天天清反而會降低效能喔！），且狀況列圖示是固定隱藏的。</p>

<p>設定手動清理時，可以選擇要不要隱藏狀況列圖示。預設值是每次清理後隱藏 30 天。圖示出現時，就是在提示你該點下去、清理收藏庫啦！</p>

<p>有些朋友想要保留極低使用率的頁面歷史，就可以勾選下方的「只整理、不清除」選項，也可調整低使用率的瀏覽次數門檻。<br /><br />
預設值是清除瀏覽過 2 次以下的紀錄，這樣可以清除掉很多幾乎無用的瀏覽歷史，對於 Firefox 運用收藏資料庫時的效率非常大！</p>

<p>版本歷史：<a href="http://code.google.com/p/placescleaner/wiki/VersionHistory">http://code.google.com/p/placescleaner/wiki/VersionHistory</a></p>
