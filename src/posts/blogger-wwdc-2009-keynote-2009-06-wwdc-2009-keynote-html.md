<p>WWDC '09（一小時前）剛結束，把在 <a href="http://www.plurk.com/p/zeddf">Plurk</a> 上筆記的一些內容轉到這來。</p>

<p>這次主要看四個網站的文字直播，中文的有 <a href="http://wwdc.hypo.cc/">hypo</a>、<a href="http://www.pcadv.com.tw/?p=8923">電腦王雜誌</a>，英文的則是 <a href="http://www.engadget.com/2009/06/08/phil-schiller-keynote-live-from-wwdc-2009/">Engadget</a> 跟 <a href="http://www.tuaw.com/2009/06/08/wwdc-2009-keynote-meta-liveblog/">TUAW</a>。現在癮科科中文版已經 <a href="http://chinese.engadget.com/2009/06/08/phil-schiller-keynote-live-from-wwdc-2009/">彙整</a> 好轉播內容，Fred 長輩則是很厲害的 <a href="http://fred.ipod.to/blog/?post/1908">完整評論</a> 了整個 Keynote。</p>

<p>本次 WWDC 中文線上轉播最大的爆點是 <a href="http://iphone4.tw/forums/kekeke.php">iPhone4.tw</a>，因為選錯聊天室作為轉播的平台，又沒把主持人的訊息區隔出來，吉米丘的發言被線上近千人完全洗掉，鄉民洗版亂版導致完全無法閱讀，右方的貼圖區居然也被貼上三點全露的<strike>東京熱</strike>A 圖，現場狀況失控，真是說有多慘烈就有多慘烈……</p>

<div id="fullpost"><p><a href="http://www.apple.com/macbookpro/">Macbook Pro</a> 系列：</p>

<p>Unibody MBP 13" &amp; 15" 終於內建 SD 卡槽，趕上五年前的技術水準了。<br>
MBP 13" 的 Firewire 回來了！終於發現拿掉是個大錯誤嘛！但是 15" 的 Express Card 不見了。MB 沒有 Express Card 我還能接受，MBP 沒有的話就…… 舊款繼續用。<br>
128GB SSD 版 13" MBP 居然要六萬大洋……<br>
忽然想到，MBP 的 BD 勒？蘋果真的打算五年後才裝 BD 啊？不能因為要大家用 iTunes 直接租、購片，就一直不給用 BD 啊……<br>
MBP 變成13" 15" 17"，那 MB 產品線打算出清完小白就丟掉了嗎囧。咦，<a href="http://www.apple.com/macbookair/">MacBook Air</a> 還沒有換成單片的觸控板，還在清庫存嗎？</p>

<p><a href="http://www.apple.com/macosx/">Snow Leopard</a> 雪豹：</p>

<p>雪豹還要到等到九月，哇靠！<br>
阿婆官方雪豹網站的 Slogan 很有自知之明：世界最先進作業系統，終於修好了。（The world's most advanced operating system, <strike>Finally</strike> Finely tuned.）<br>
修好的東西可真多：時光機變快、Finder 重寫、加入無線網路也增快。Preview PDF 跨多欄選取內容不會再混在一起了，聰明聰明。<br>
Safari 4 正式版今天上場，贊耶（雖然大多都還是用 Firefox）<br>
多重觸控機種通通開放三指跟四指了，這點還算有良心。<br>
雪豹終於拋棄 PowerPC 而去了，我的 <a href="http://www.flickr.com/photos/irvin/1552889110/">小粉</a> 沒得升級啦。看來就是把 Power PC 的 Code 丟掉，才能比豹子瘦 6GB的。<br>
iPhone 的中文手寫功能移植到雪豹上了，不過說真的，對台灣用戶不是很實用，應該是著眼於外國人而設計的。<br>
每次看 Apple 的 Keynote 到後來都會很想來寫 Xcode 程式……感覺 API 都很威 (LOL)<br>
OS X 第一次出現升級版，破盤價 USD $29，我覺得因為雪豹 = Patched 豹子，才會有這種價錢吧。Anyway，我還是會用快樂版，這是原則問題。買了 Apple 的硬體，每次都要另外加購 OS 升級，就是很不爽。</p>

<p><a href="http://www.apple.com/iphone/">iPhone</a>：</p>

<p>iPhone 全系列免費升級 OS 3.0，iPod touch 又是 $9.95，Anyway 繼續快樂去。<br>
OS 3.0 的 Notes 終於可以同步了！這種 1.0 早該有卻忘記寫的功能，現在才補回來。<br>
等待已久的 Tethering 功能終於出現了。種花電信應該不會笨到不開放吧？<br>
Find My iPhone 綁在 mobile me 裡面，靠，實用是很實用，但是這東西一年就比雪豹貴了。<br>
In-App Purchase，以後玩賽車要開新車？0.99。玩 Sims 要新傢俱？0.99…… 居然馬上就端出 Asphlt 5，真的要花錢買新車了。有沒有 Asphlt 4 的升級價啊？<br>
在展示 Stanza 的電子書功能，不知道哪天蘋果日報才會想通，跳進來做 iPhone Apps？對其他「質報」不期不待了。<br>
Tom Tom! GPS 軟體出現！有多國地圖，不知道會不會有中文的台灣地圖可用。</p>

<p>新 iPhone 一直沒出來，要不是 One more thing 的話，就是留給jobs月底上台秀嘍？中間這一堆 Apps 秀沒啥有趣的……</p>

<p><a href="http://www.apple.com/iphone/iphone-3g-s/">iPhone 3G S</a>：</p>

<p>靠！iPhone 3G 真的來小改款。iPhone 3G S 台灣八月上市，還有一個月，種花電信要是 3G 庫存沒清光，就哭哭了。這次至少有時間表，不會拖半年才上市了。<br>
iPhone 3G 是 3.6M，3G S 是 7.2M，一樣都是 HSDPA，差真多 QQ<br>
錄影只在 3G S 上提供，會不會太過分！果然是蘋果作風，不意外。大概 OS 4.0 就會丟掉舊的 2G 跟 3G iPhone 了。不要跟我說 3G 也沒有語音控制啊！<br>
3G S 也有電子羅盤了，果然 Andriod 上面有的都加上去了，我也想要能 <a href="http://google-latlong.blogspot.com/2009/05/sky-map-for-android-location-and.html">轉來轉去的星圖</a> 啊。<br>
iPhone 3G 8GB 版本售價立刻降三千，變成 USD $99。胡亂預測種花電信的 iPhone 售價，可能會變成：通通都預繳一萬三、綁 1049 兩年，3G 8GB 購機一萬；3G S 16GB 購機一萬四；3G S 32GB 購機一萬七。</p>

<p>One more thing, WWDC 該出來的都出來了，那月底 Steve Jobs 的復出秀要秀啥啊，傳說中的 iPhone 3G L(ite)？</p>
