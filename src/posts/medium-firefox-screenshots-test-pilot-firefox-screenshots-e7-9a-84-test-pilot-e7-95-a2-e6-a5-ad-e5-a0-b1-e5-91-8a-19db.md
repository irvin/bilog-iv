### Firefox Screenshots 的 Test Pilot 畢業報告

> Page Shot 是最受歡迎的 Test Pilot 實驗項目，每日使用率高過其他功能兩倍以上；分享的流量持續成長，且也吸引非 Firefox 的用戶觀看。

**Screenshots is Shipping in Firefox 56**
*Let’s start with the big news: we just shipped Firefox Screenshots with Firefox 56. Screenshots began life as the Test…*medium.com

經過 Test Pilot 測試的 Firefox 網頁抓圖分享功能「[Page Shot](https://testpilot.firefox.com/experiments/page-shot/)」，正式在 Firefox 56 中以「[Firefox Screenshots](https://screenshots.firefox.com/)」名稱釋出。本文說明了 Test Pilot 時期的研究成果，與其在正式版中的功能差異。
